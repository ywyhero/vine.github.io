安装node环境

brew install node
也可去node官网（中文网：http://nodejs.cn/ 英文网：https://nodejs.org/en/）下载安装包自行安装。

先创建一个服务 (server.js)

let http = require(‘http’);
functon start () {
function onRequest (request, response) {
response.writeHead(200, {‘Content-Type’: ‘text/plain’});
response.write(‘Hello World’);
response.end();
}
http.creatServer(onRequest).listen(8888);
console.log(‘服务已开启’);
}
start() //执行程序

注:直接打开终端，跳转到当前文件夹输入node server终端应显示”服务已开启“。浏览器打开 http://localhost:8888 应显示“Hello World”。

微信开发缩略图不显示的问题解决方法

发表于 2017-05-12

JS-SDK报错 invalid signature签名错误

  • 1、确认config中nonceStr（js中驼峰标准大写S）, timestamp与用以签名中的对应noncestr, timestamp一致。

  • 2、确认 config 中的 appid 与用来获取 jsapi_ticket 的 appid 一致。

图片类型的错误

  • 1、图片不能是base64类型的。

  • 2、创建一个img标签，在body最前面
    <img style="width: 0; heigth: 0; position:absolute; left: 2000px" src="XXX.jpg">

微信官方

  • config 必须填写的内容

wx.config({
debug: true, // 开启调试模式,调用的所有api的返回值会在客户端alert出来，若要查看传入的参数，可以在pc端打开，参数信息会通过log打出，仅在pc端时才会打印。
appId: '', // 必填，公众号的唯一标识
timestamp: , // 必填，生成签名的时间戳
nonceStr: '', // 必填，生成签名的随机串
signature: '',// 必填，签名，见附录1

});

解决iframe不支持ios的方法

发表于 2017-05-08

html布局

<div id="iframe-box">
<iframe id="ifram" width="100%"></iframe>
</div>

css样式

#ifram {
border: 0;
width: 1px;
min-width: 100%;
*width: 100%;
}

js逻辑

　　
let ifram = document.getElementById(‘ifram’);
if (navigator.userAgent.match(/iPad|iPhone/i)) {
let iframe_box = document.getElementById(‘iframe-box’);
iframe_box.style.width = 100 + ‘%’;
iframe_box.style.overflowX = ‘hidden’;
iframe_box.style.overflowY = ‘scroll’;
iframe_box.style.webkitOverflowScrolling = ‘touch’;
ifram.setAttribute(‘scrolling’, ‘no’);
iframe_box.appendChild(ifram)
}

hexo换肤

发表于 2017-05-08

更换为hexo主题

  • 进入hexo文件夹

    cd your-hexo-site

  • 克隆next代码并存放到 themes/next 目录下

    git clone https://github.com/iissnan/hexo-theme-next themes/next

  • 在站点 _config.yml 中修改:

    theme: next

设置头像

  • 在站点 _config.yml 中添加

    avatar: http://…. # 头像的URL

添加分类

  • 新建一个页面，命名为 categories

    hexo new page categories

  • 在 source/categories 目录的 index.md 中修改:

    title: 分类
    date: 2015-12-02 12:44:45
    type: “categories”

  • 在主题的 _config.yml 中取消注释:

    categories: /categories

  • 在要分类的文章中加入 category 属性:

    category: “hexo” # 分类为 hexo

添加标签

  • 新建一个页面，命名为 tags

    hexo new page tags

  • 在 source/tags 目录的 index.md 中修改:

    title: tags
    date: 2015-12-02 12:10:34
    type: “tags”

  • 在主题的 _config.yml 中取消注释:

    tags: /tags

  • 在需要变迁的文章中加入 tgs 属性:

    tags: [spark,sparkSQL] # 添加spark和sparkSQL两个标签

添加评论

  • 在 多说 创建一个站点，创建时填写的多说域名就是你的 duoshuo_shortname

  • 在站点 _config.yml 中设置：

    duoshuo_shortname: your_duoshuo_shortname
