发表于 2017-06-02 | 分类于 工作问题

安卓focus输入框时页面被压缩的解决办法

原理：focus时页面宽度等于屏幕宽度 blur时页面宽度设为100%或者auto

setFocusHeight () {
let login = document.querySelector(‘.login’);
let height = window.screen.height;
login.style.height = height + ‘px’
},
setBlurHeight() {
let login = document.querySelector(‘.login’);
login.style.height = ‘100%’
}

ios button按钮有默认样式的解决办法

input[type=”button”], input[type=”submit”], input[type=”reset”] {
-webkit-appearance: none;
}

ios不兼容iframe的解决方法

html布局

<div id="iframe-box">
<iframe id="ifram" width="100%"></iframe>
</div>

css样式

#ifram {
border: 0;
width: 1px;
min-width: 100%;
*width: 100%;
}

js逻辑

　　
let ifram = document.getElementById(‘ifram’);
if (navigator.userAgent.match(/iPad|iPhone/i)) {
let iframe_box = document.getElementById(‘iframe-box’);
iframe_box.style.width = 100 + ‘%’;
iframe_box.style.overflowX = ‘hidden’;
iframe_box.style.overflowY = ‘scroll’;
iframe_box.style.webkitOverflowScrolling = ‘touch’;
ifram.setAttribute(‘scrolling’, ‘no’);
iframe_box.appendChild(ifram)
}
