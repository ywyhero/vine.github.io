es6去重的办法

发表于 2017-09-01 | 分类于 学习日志

在学习node的时候，有个需求是将数据从数据中获取并去重，得到同事的帮助得知es6的set方便，在此记录一下：

方法一：

let items = new Set([1,2,3,3,2,5]) //返回的数据是 Set(4) {1,2,3,5}
let arry = Array.from(items) //返回的数据是[1,2,3,5]

方法二：

let s = new Set();
s.add(1);
s.add(2);
s.add(2);
//返回数据是Set(2) {1, 2}
let arry = Array.from(s)
