前端点击复制的解决方案

发表于 2017-07-27 | 分类于 工作问题

某天突然来了一个点击复制到剪切板的需求，之前没有做过这类的，于是在网上找了一大推解决方案。

  1. 用js原生写的（浏览器兼容是很大的问题）
  2. 复制框架（ZeroClipboard,jquery.zclip,clipboard等）

由于兼容问题js原生的就不考虑了，框架中，jquery.zclip需要引进jquery，所以也不再考虑范围之内了。

ZeroClipboard，clipboard两个框架都是比较方便的。小伙伴们可以根据自己的爱好随心挑选。

使用方法很简单：（本人用的是clipboard，所以简单介绍下这个。ZeroClipboard可以自行到官网查看用法。）

  1. npm install clipboard

  2. 点击按钮必须需要 data-clipboard-target=“”和 data-clipboard-action=”” 自定义属性 如：

<button class="btn" data-clipboard-action="copy" data-clipboard-target="#bar"> Cut to clipboard </button>

复制成功和失败后的操作：

clipboard.on(‘success’, function(e) {
console.info(‘Action:’, e.action);
console.info(‘Text:’, e.text);
console.info(‘Trigger:’, e.trigger);

e.clearSelection();

});

clipboard.on(‘error’, function(e) {
console.error(‘Action:’, e.action);
console.error(‘Trigger:’, e.trigger);
});
